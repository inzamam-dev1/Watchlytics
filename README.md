# NetflixGPT 

A Netflix-inspired movie browsing app with AI-powered search.

## Features
- Browse Popular, Top Rated & Now Playing movies
- Trailer autoplay
- Redirect to official Netflix page
- Firebase Authentication

## Tech Stack
- React
- Redux Toolkit
- Tailwind CSS
- Firebase
- TMDB API

## Setup
1. Clone repo
2. npm install
3. Add .env file
4. npm run dev
